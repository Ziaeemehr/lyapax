# Getting support

- **Usage questions** ("how do I model X with lyapax", "why do my
  exponents look wrong"): open a
  [GitHub Discussion](https://github.com/Ziaeemehr/lyapax/discussions).
- **Bugs and crashes**: open a
  [GitHub issue](https://github.com/Ziaeemehr/lyapax/issues) using the bug
  report template.
- **Feature requests**: open a
  [GitHub issue](https://github.com/Ziaeemehr/lyapax/issues) using the
  feature request template.
- **Security or conduct concerns**: email a.ziaeemehr@gmail.com directly
  rather than opening a public issue.

Before asking a usage question, check whether it's already covered in:

- [`README.md`](README.md) for the minimal example and scope
- [`docs/background/capabilities.md`](docs/background/capabilities.md)
  for what lyapax can and cannot do
- the [`examples/`](examples/) gallery for a worked example close to your
  use case
